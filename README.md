# Information sur les ateliers
Ce petit guide est içi juste à titre indicatif pour peut-être aider à une impulsion.

* Objectif d'une durée ~ 2 heures.
* Un sujet maitrisé pour une fluiditée dans les interactions.

>Documenter au maximum l’atelier avec peut-être un dossier partagé, dans lequel vous pouvez réunir vos documents, tutoriels, photos... Une tiers personne pourrait y contribuer par la suite :octo:

# Construire un plan/fiche technique.
Ceci sera très utile peut être pour vous lors de l'atelier, mais surtout pour de potentiel future contributeurs ou pour reprendre votre atelier.

Ci-dessous un exemple pour faire une fiche technique.

---

### Les possibles Pré requis

* *Il y a t'il des choses particulières à préparer*
* *peut-être du materiels spécifiques*
* *faciliter la communication, les copier/coller ou click de liens avec un fichier de partage type "Slide" avec par ex:  [Bunkr](https://bunkrapp.com/), [Framapad](https://framapad.org/) ou un client IRC. ex: [Hexchat](https://hexchat.github.io/downloads.html) / [mIRC](http://www.mirc.com/get.html).*

Expliquer le tarif de la contribution financière si il y a.
NB: les contributions sont a déstination de l’association et permet de financer l’achats de machine et l’organisation d’événements.

### Titre de la l'atelier


**étape, temps, détails**

```yaml
titre: Atelier soudure sur Arduino,
# pré requis?
10min: Introduction présentation,
15min: description des type de soudure et les erreurs communes,
15min: souder son Adruino mini ou autre,
20min: ratraper une erreur de soudure,
30min: test des arduino soudé/branché,
30min: prise de photos commentaires questions réponses,
```
Nous vous conseillons de créer un dossier sur partagable, dans lequel vous pouvez réunir tous les documents, tutoriels, photos pour documenter au maximum l'atelier, dans une optique de partage de la connaissance.

### aide au support visuel
Même si votre atelier n'a pas recoure à l'informatique, il est souhaitable que l'on en garde une trace documentée dans nos répertoires partagés.
Il existe sur internet beaucoup de solutions gratuites pour organiser des activités. Nous vous conseillons d'utiliser des applications ou logiciel le plus libre possible. Mais rester fonctionnel.

* faire des slides rapides avec [Bunkr](https://bunkrapp.com/)
* un document partagable [Framapad](https://framapad.org/)
